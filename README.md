# hiit-timer
Intervaltimer for performing HIIT(High Intensity Interval Training).

## install
npm i hiit-timer

## Usage
1.Execute the command.

`$ node hiit-timer`

2.select workout time

```
? select workout time
> 10
  20
  30
  ...
```

3.select rest time
```
? select rest time
> 10
  20
  30
  ...
```

4.select set number
```
? select set number
> 1
  2
  3
  ...
```
